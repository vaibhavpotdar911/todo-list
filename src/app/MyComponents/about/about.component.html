To-do List!

This is a simple To-do list to make a list of some tasks which are supposed to be completed!
This application is commited on Github and has a CI/CD Pipeline using AWS CodePipeline which automatically fetches the code from Github, builds the application using Amazon CodeBuild and deploys the app using CodeDeploy. This application is hosted using Amazon S3!

This is a single page application built on Angular 11 and Bootstrap.
